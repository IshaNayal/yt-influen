# Scripts

This folder contains utility scripts for data verification and summaries.

## Files

- **check_all_bigrams.py** - Verifies bigram extraction methodology (corpus-wide approach)
- **create_bigram_summary.py** - Creates summary of bigram analysis results

## Usage

These scripts were used for intermediate data validation and verification during the analysis pipeline.

## Related Folders

- Main analysis scripts are in `/analysis/`
- Data collection scripts are in `/data_collection/`
